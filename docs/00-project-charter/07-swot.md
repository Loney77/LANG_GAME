# SWOT-анализ

Анализ текущего состояния предметной области и проектного решения.

| | **Полезное** | **Вредное** |
|---|---|---|
| **Внутренние факторы** | **Strengths (Сильные стороны)** | **Weaknesses (Слабые стороны)** |
| | • Уникальная ниша: цифровых игр для карачаевского языка почти нет<br>• Игровая форма повышает вовлечённость и удержание<br>• Серверная валидация защищает от подсказок и держит контент централизованно<br>• Готовый токенизатор алфавита (диграфы) - техническое преимущество | • Малый стартовый словарь (~246 слов, только а/б)<br>• Зависимость от носителей языка для качества контента<br>• Нет автогенерации больших объёмов контента |
| **Внешние факторы** | **Opportunities (Возможности)** | **Threats (Угрозы)** |
| | • Поддержка инициатив по сохранению языков народов РФ<br>• Расширение на другие игры и уровни<br>• Потенциал коллективного пополнения словаря сообществом<br>• Масштабирование на другие тюркские языки | • Ограниченная аудитория носителей языка<br>• Сложность поиска лицензионно чистых словарных источников<br>• Конкуренция с универсальными платформами (Duolingo и т.п.)<br>• Риск низкой активности пользователей |

## Стратегические выводы (TOWS)

- **S×O:** использовать уникальность ниши и игровую механику для привлечения
  сообщества к пополнению словаря.
- **W×T:** снизить зависимость от объёма контента - сделать админ-CRUD простым и
  предусмотреть импорт из существующих словарей.
- **S×T:** качество и достоверность (проверка носителями) как ответ на
  конкуренцию универсальных платформ.
