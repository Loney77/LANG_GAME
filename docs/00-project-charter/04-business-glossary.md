# Бизнес-глоссарий

Единая терминология предметной области (20 терминов). На Этапе 1 глоссарий
расширяется системными терминами.

| # | Термин | Определение |
|---|--------|-------------|
| 1 | **Карачаевский язык** | Тюркский язык карачаево-балкарской группы; письменность — кириллица с диграфами. |
| 2 | **Диграф** | Сочетание двух букв, обозначающее один звук и считающееся одной буквой алфавита: **гъ, къ, нъ, нг, дж**. |
| 3 | **Слово (словарная статья)** | Единица словаря: карачаевское слово, его русский перевод, тема, длина в буквах. |
| 4 | **Перевод** | Краткое русское значение слова, используемое в играх (викторина, подсказки). |
| 5 | **Тема** | Тематическая категория слов (числа, цвета, семья, еда, животные…). |
| 6 | **Игра** | Тип игрового механизма: Сёздл, анаграмма, викторина, кроссворд. |
| 7 | **Сёздл** | Игра на угадывание слова за ограниченное число попыток с подсветкой букв. |
| 8 | **Анаграмма** | Игра: из перемешанных букв слова нужно собрать исходное слово. |
| 9 | **Викторина** | Игра «вопрос — варианты ответа» с одним верным вариантом. |
| 10 | **Кроссворд** | Игра по заполнению сетки словами по горизонтали/вертикали. |
| 11 | **Задание (Puzzle)** | Конкретный экземпляр игры с заданными данными (слово-цель, набор букв, вопросы, сетка). |
| 12 | **Ежедневное задание (Daily challenge)** | Одно задание дня, единое для всех игроков; основа честного лидерборда. |
| 13 | **Попытка / Сессия игры** | Один проход игроком конкретного задания с результатом (победа/поражение). |
| 14 | **Подсветка букв** | Обратная связь Сёздл: 🟩 на месте, 🟨 есть в слове, ⬜ отсутствует. |
| 15 | **Очки (Score)** | Числовая оценка результата игры (зависит от попыток и времени). |
| 16 | **Лидерборд** | Рейтинг игроков по сумме очков за период / по типу игры. |
| 17 | **Игрок** | Пользователь приложения, обучающийся; роль `USER`. |
| 18 | **Администратор** | Пользователь, управляющий словарём и контентом; роль `ADMIN`. |
| 19 | **Носитель языка** | Эксперт, проверяющий достоверность переводов и вопросов. |
| 20 | **Оффлайн-режим** | Работа приложения без сети за счёт локального кэша словаря и заданий. |
