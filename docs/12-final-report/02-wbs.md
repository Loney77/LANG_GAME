# Структура декомпозиции работ (WBS)

Иерархическая декомпозиция работ проекта до уровня рабочих пакетов.

## 1. Иерархия работ

```
1. Мобильное приложение «Сёз оюн»
├── 1.1. Инициация и бизнес-анализ
│   ├── 1.1.1. Паспорт проекта
│   ├── 1.1.2. Бизнес-контекст и прецеденты (IDEF0, BUC)
│   └── 1.1.3. Глоссарий, стейкхолдеры, SWOT
├── 1.2. Анализ требований
│   ├── 1.2.1. Модель прецедентов и доменная модель
│   ├── 1.2.2. Спецификации прецедентов
│   └── 1.2.3. Матрица трассировки
├── 1.3. Архитектура
│   ├── 1.3.1. Диаграмма пакетов PCMEF
│   ├── 1.3.2. Спецификация интерфейсов
│   └── 1.3.3. Архитектурные решения (ADR)
├── 1.4. Проектирование базы данных
│   ├── 1.4.1. ER-диаграмма
│   ├── 1.4.2. DDL-скрипты (3НФ, ограничения)
│   └── 1.4.3. Стратегия ORM-маппинга
├── 1.5. Детальное проектирование
│   ├── 1.5.1. Диаграммы последовательностей
│   ├── 1.5.2. Диаграмма классов проектирования
│   └── 1.5.3. Спецификации методов
├── 1.6. Реализация ядра (сервер)
│   ├── 1.6.1. Слои Entity и Foundation
│   ├── 1.6.2. Бизнес-логика игр (Mediator)
│   ├── 1.6.3. REST-контроллеры и безопасность
│   └── 1.6.4. Модульные тесты и отчёт покрытия
├── 1.7. Рефакторинг
│   ├── 1.7.1. Паттерн Data Mapper
│   ├── 1.7.2. Identity Map
│   └── 1.7.3. Статический анализ и доработка тестов
├── 1.8. Реализация интерфейса (клиент)
│   ├── 1.8.1. Навигация и экраны
│   ├── 1.8.2. Интеграция с API и обработка состояний
│   └── 1.8.3. Оффлайн-кэш
└── 1.9. Завершение
    ├── 1.9.1. Техническое задание и оценки (WBS, Гант, COCOMO)
    ├── 1.9.2. Руководства пользователя и администратора
    ├── 1.9.3. Пояснительная записка
    └── 1.9.4. Презентация к защите
```

## 2. Словарь рабочих пакетов

| Код | Рабочий пакет | Результат |
|-----|---------------|-----------|
| 1.1 | Инициация | Паспорт проекта, бизнес-модели, глоссарий |
| 1.2 | Требования | Use Case, доменная модель, трассировка |
| 1.3 | Архитектура | Пакеты PCMEF, интерфейсы, ADR |
| 1.4 | База данных | ER-диаграмма, DDL, миграции Flyway |
| 1.5 | Детальное проектирование | Sequence- и class-диаграммы, спецификации |
| 1.6 | Реализация ядра | Серверный код, тесты, отчёт JaCoCo |
| 1.7 | Рефакторинг | Внедрённые паттерны, отчёт анализа кода |
| 1.8 | Интерфейс | Мобильный клиент, интеграция, кэш |
| 1.9 | Завершение | Итоговая документация и презентация |
