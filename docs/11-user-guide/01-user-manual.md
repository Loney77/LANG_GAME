# Руководство пользователя

## 1. Назначение приложения

«Сёз оюн» — мобильное приложение для изучения карачаевского языка в игровой форме.
Приложение предлагает четыре словесные игры на карачаевской лексике, ведёт счёт
очков и общий рейтинг игроков.

## 2. Требования к устройству

- Android 8.0 и выше.
- Подключение к интернету для входа, синхронизации результатов и получения заданий.
  Словарь и текущие задания кэшируются для просмотра без сети.

## 3. Начало работы

### 3.1. Регистрация

1. Запустите приложение — откроется экран входа.
2. Нажмите «Зарегистрироваться».
3. Введите имя пользователя, адрес электронной почты и пароль.
4. После успешной регистрации вход выполняется автоматически.

### 3.2. Вход

1. На экране входа введите имя пользователя и пароль.
2. Нажмите «Войти». При успешной аутентификации откроется главный экран.

Сессия сохраняется на устройстве: повторный вход при следующих запусках не требуется
до выхода из учётной записи.

## 4. Главный экран

Главный экран содержит меню выбора игры и переходы к рейтингу и профилю:

| Раздел | Назначение |
|--------|-----------|
| Сёздл | Угадывание слова по подсветке букв |
| Анаграммы | Сборка слова из перемешанных букв |
| Викторина | Выбор правильного перевода из вариантов |
| Кроссворд | Заполнение пересекающихся слов в сетке |
| Лидерборд | Рейтинг игроков по сумме очков |
| Профиль | Данные учётной записи и выход |

## 5. Игры

### 5.1. Сёздл

Нужно угадать загаданное слово из пяти букв за ограниченное число попыток.
После каждой попытки буквы подсвечиваются:

- зелёный — буква на своём месте;
- жёлтый — буква есть в слове, но на другой позиции;
- серый — буквы нет в слове.

Диграфы карачаевского алфавита (гъ, къ, нъ, нг, дж) считаются одной буквой и
занимают одну ячейку.

### 5.2. Анаграммы

Из набора перемешанных букв нужно собрать исходное слово. Буквы переносятся
нажатием из общего набора в строку ответа и обратно. После сборки нажмите кнопку
проверки.

### 5.3. Викторина

Показывается слово на карачаевском языке и несколько вариантов перевода.
Нужно выбрать правильный вариант. Засчитывается верный ответ.

### 5.4. Кроссворд

В сетке размещены пересекающиеся слова. Выберите ячейку, введите буквы слова и
подтвердите. Правильно заполненные слова фиксируются.

## 6. Очки и рейтинг

За завершённые игры начисляются очки с учётом числа попыток и затраченного времени.
Сумма очков формирует позицию в лидерборде. Рейтинг можно отфильтровать по виду игры
и периоду.

## 7. Профиль и выход

В разделе «Профиль» отображаются данные учётной записи. Для завершения сеанса
нажмите «Выйти» — потребуется повторный вход при следующем запуске.

## 8. Работа без сети

При отсутствии подключения приложение использует кэшированные словарь и задания.
Результаты, полученные офлайн, отправляются на сервер после восстановления связи.
